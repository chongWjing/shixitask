## 12.2 商品管理页面

> "商品管理是后台最常用的功能。管理员需要看到所有商品的列表，能新增商品、编辑商品信息、删除商品、控制商品上下架。我们用Element Plus的Table、Pagination、Dialog、Form组件来实现。"

### 步骤1：创建商品管理页面

创建文件 `frontend/src/views/admin/ProductManage.vue`：

```vue
<template>
  <!-- 商品管理页面 -->
  <div>
    <!-- 工具栏：搜索 + 新增 -->
    <el-card shadow="never" style="margin-bottom: 16px; border-radius: 12px">
      <div style="display: flex; justify-content: space-between; align-items: center">
        <el-input
          v-model="searchKeyword"
          placeholder="搜索鲜花商品..."
          style="width: 240px"
          clearable
          @input="filterProducts"
        >
          <template #prefix>
            <el-icon><Search /></el-icon>
          </template>
        </el-input>
        <el-button type="primary" @click="showAddDialog" style="background-color: #C71526; border-color: #C71526;">
          <el-icon><Plus /></el-icon> 新增商品
        </el-button>
      </div>
    </el-card>

    <!-- 商品表格 -->
    <el-card shadow="never" style="border-radius: 12px">
      <el-table :data="pagedProducts" stripe style="width: 100%">
        <el-table-column prop="id" label="ID" width="60" />
        <el-table-column prop="name" label="商品名" min-width="200" />
        <el-table-column label="价格" width="100">
          <template #default="{ row }">
            ¥{{ row.price }}
          </template>
        </el-table-column>
        <el-table-column prop="stock" label="库存" width="80" />
        <el-table-column prop="sales" label="销量" width="80" />
        <el-table-column prop="category" label="分类" width="90" />
        <el-table-column label="状态" width="80">
          <template #default="{ row }">
            <el-tag :type="row.status === 1 ? 'success' : 'info'" size="small">
              {{ row.status === 1 ? '上架' : '下架' }}
            </el-tag>
          </template>
        </el-table-column>
        <el-table-column label="操作" width="240" fixed="right">
          <template #default="{ row }">
            <el-button size="small" type="primary" link @click="showEditDialog(row)">
              编辑
            </el-button>
            <el-button size="small" type="danger" link @click="handleDelete(row)">
              删除
            </el-button>
            <el-button
              v-if="row.status === 1"
              size="small" type="warning" link
              @click="handleToggleStatus(row, 0)"
            >
              下架
            </el-button>
            <el-button
              v-if="row.status !== 1"
              size="small" type="success" link
              @click="handleToggleStatus(row, 1)"
            >
              上架
            </el-button>
          </template>
        </el-table-column>
      </el-table>

      <!-- 分页 -->
      <div style="display: flex; justify-content: space-between; align-items: center; margin-top: 16px">
        <span style="font-size: 13px; color: #666">
          共 {{ filteredProducts.length }} 条记录
        </span>
        <el-pagination
          v-model:current-page="currentPage"
          :page-size="pageSize"
          :total="filteredProducts.length"
          layout="prev, pager, next"
          small
        />
      </div>
    </el-card>

    <!-- 新增/编辑商品弹窗 -->
    <el-dialog
      v-model="dialogVisible"
      :title="isEdit ? '编辑商品' : '新增商品'"
      width="500px"
    >
      <el-form ref="productFormRef" :model="productForm" :rules="productRules" label-width="100px">
        <el-form-item label="商品名称" prop="name">
          <el-input v-model="productForm.name" placeholder="请输入商品名称" />
        </el-form-item>
        <el-form-item label="商品价格" prop="price">
          <el-input-number v-model="productForm.price" :min="0.01" :precision="2" style="width: 100%" />

        </el-form-item>
        <el-form-item label="商品分类" prop="category">
          <el-select v-model="productForm.category" placeholder="请选择分类" style="width: 100%">
            <el-option label="鲜花" value="鲜花" />
            <el-option label="永生花" value="永生花" />
            <el-option label="花束" value="花束" />
            <el-option label="花篮" value="花篮" />
            <el-option label="绿植" value="绿植" />
            <el-option label="礼品花" value="礼品花" />

        </el-form-item>
        <el-form-item label="商品图片URL">
          <el-input v-model="productForm.image" placeholder="请输入图片地址" />

        <el-button @click="dialogVisible = false">取消</el-button>

      </template>
    </el-dialog>
  </div>
</template>

<script>
import {
  getAdminProductList,
  addProduct,
  updateProduct,
  deleteProduct,
  updateProductStatus
} from '../../api/admin'

export default {
  name: 'ProductManage',
  data() {
    return {
      // 所有商品数据
      allProducts: [],
      // 搜索关键词
      searchKeyword: '',
      // 过滤后的商品
      filteredProducts: [],
      // 分页
      currentPage: 1,
      pageSize: 10,
      // 弹窗
      dialogVisible: false,
      isEdit: false,
      // 商品表单
      productForm: {
        id: null,
        name: '',
        price: null,
        stock: null,
        category: '鲜花',
        image: '',
        description: ''
      },
      // 表单验证规则
      productRules: {
        name: [{ required: true, message: '请输入商品名称', trigger: 'blur' }],
        price: [{ required: true, message: '请输入商品价格', trigger: 'blur' }],
        stock: [{ required: true, message: '请输入库存数量', trigger: 'blur' }],
        category: [{ required: true, message: '请选择商品分类', trigger: 'change' }]
      }
    }
  },
  computed: {
    // 当前页数据
    pagedProducts() {
      const start = (this.currentPage - 1) * this.pageSize
      return this.filteredProducts.slice(start, start + this.pageSize)
    }
  },
  methods: {
    // 加载商品列表
    async loadProducts() {
      try {
        const result = await getAdminProductList()
        if (result.code === 200) {
          this.allProducts = result.data || []
          this.filterProducts()
        } else {
          this.$message.error('加载商品列表失败：' + (result.message || ''))
        }
      } catch (error) {
        console.error('加载商品列表失败：', error)
        this.$message.error('网络错误，请稍后重试')
      }
    },

    // 搜索过滤
    filterProducts() {
      const keyword = this.searchKeyword.trim().toLowerCase()
      if (keyword) {
        this.filteredProducts = this.allProducts.filter(p =>
          p.name && p.name.toLowerCase().includes(keyword)
        )
      } else {
        this.filteredProducts = [...this.allProducts]
      }
      this.currentPage = 1
    },

    // 显示新增弹窗
    showAddDialog() {
      this.isEdit = false
      this.productForm = {
        id: null,
        name: '',
        price: null,
        stock: null,
        category: '鲜花',
        image: '',
        description: ''
      }
      this.dialogVisible = true
    },

    // 显示编辑弹窗
    showEditDialog(row) {
      this.isEdit = true
      this.productForm = {
        id: row.id,
        name: row.name || '',
        price: row.price,
        stock: row.stock,
        category: row.category || '鲜花',
        image: row.image || '',
        description: row.description || ''
      }
      this.dialogVisible = true
    },

    // 保存商品（新增或编辑）
    async handleSave() {
      try {
        await this.$refs.productFormRef.validate()
      } catch {
        return
      }

      try {
        let result
        if (this.isEdit) {
          result = await updateProduct(this.productForm)
        } else {
          result = await addProduct(this.productForm)
        }

        if (result.code === 200) {
          this.$message.success(this.isEdit ? '商品修改成功' : '商品添加成功')
          this.dialogVisible = false
          this.loadProducts()
        } else {
          this.$message.error(result.message || '操作失败')
        }
      } catch (error) {
        console.error('保存商品失败：', error)
        this.$message.error('网络错误，请稍后重试')
      }
    },

    // 删除商品
    async handleDelete(row) {
      try {
        await this.$confirm(`确定要删除商品"${row.name}"吗？此操作不可恢复！`, '确认删除', {
          confirmButtonText: '确认删除',
          cancelButtonText: '取消',
          type: 'warning'
        })

        const result = await deleteProduct(row.id)
        if (result.code === 200) {
          this.$message.success('商品删除成功')
          this.loadProducts()
        } else {
          this.$message.error(result.message || '删除失败')
        }
      } catch {
        // 用户取消
      }
    },

    // 上架/下架
    async handleToggleStatus(row, status) {
      const statusText = status === 1 ? '上架' : '下架'
      try {
        const result = await updateProductStatus({ id: row.id, status })
        if (result.code === 200) {
          this.$message.success(`商品${statusText}成功`)
          this.loadProducts()
        } else {
          this.$message.error(result.message || `${statusText}失败`)
        }
      } catch (error) {
        console.error(`${statusText}失败：`, error)
        this.$message.error('网络错误，请稍后重试')
      }
    }
  },
  created() {
    this.loadProducts()
  }
}
</script>
```

> "注意商品分类选项已更新为花之恋鲜花电商的分类：鲜花、永生花、花束、花篮、绿植、礼品花。新增商品弹窗的默认分类为'鲜花'，保存按钮使用主题色 #C71526。"

### 步骤2：创建订单管理页面

创建文件 `frontend/src/views/admin/OrderManage.vue`：

```vue
<template>
  <!-- 订单管理页面 -->
  <div>
    <!-- 工具栏：状态筛选 + 刷新 -->
    <el-card shadow="never" style="margin-bottom: 16px; border-radius: 12px">
      <div style="display: flex; justify-content: space-between; align-items: center">
        <div style="display: flex; align-items: center; gap: 8px">
          <span style="font-size: 14px; color: #666">状态筛选：</span>
          <el-select v-model="statusFilter" placeholder="全部" style="width: 160px" @change="filterOrders">
            <el-option label="全部" value="" />
            <el-option label="待付款" :value="0" />
            <el-option label="已付款" :value="1" />
            <el-option label="已发货" :value="2" />
            <el-option label="已完成" :value="3" />
            <el-option label="已取消" :value="4" />
          </el-select>
        </div>
        <el-button @click="loadOrders" style="background-color: #C71526; border-color: #C71526; color: #fff;">
          <el-icon><Refresh /></el-icon> 刷新
        </el-button>
      </div>
    </el-card>

    <!-- 订单表格 -->
    <el-card shadow="never" style="border-radius: 12px">
      <el-table :data="pagedOrders" stripe style="width: 100%">
        <el-table-column prop="orderNo" label="订单号" width="200" />
        <el-table-column prop="productName" label="商品名" min-width="180" />
        <el-table-column prop="quantity" label="数量" width="70" />
        <el-table-column label="总价" width="100">
          <template #default="{ row }">
            ¥{{ row.totalPrice }}
          </template>
        </el-table-column>
        <el-table-column label="状态" width="90">
          <template #default="{ row }">
            <el-tag :type="statusTypeMap[row.status]" size="small">
              {{ statusTextMap[row.status] || '未知' }}
            </el-tag>
          </template>
        </el-table-column>
        <el-table-column prop="receiverName" label="收货人" width="90" />
        <el-table-column label="时间" width="160">
          <template #default="{ row }">
            {{ formatTime(row.createTime) }}
          </template>
        </el-table-column>
        <el-table-column label="操作" width="120" fixed="right">
          <template #default="{ row }">
            <el-button
              v-if="row.status === 1"
              size="small" type="primary" link
              @click="handleUpdateStatus(row, 2)"
            >
              发货
            </el-button>
            <el-button
              v-if="row.status === 2"
              size="small" type="success" link
              @click="handleUpdateStatus(row, 3)"
            >
              完成
            </el-button>
            <span v-if="row.status !== 1 && row.status !== 2" style="color: #999; font-size: 12px">--</span>
          </template>
        </el-table-column>
      </el-table>

      <!-- 分页 -->
      <div style="display: flex; justify-content: space-between; align-items: center; margin-top: 16px">
        <span style="font-size: 13px; color: #666">
          共 {{ filteredOrders.length }} 条记录
        </span>
        <el-pagination
          v-model:current-page="currentPage"
          :page-size="pageSize"
          :total="filteredOrders.length"
          layout="prev, pager, next"
          small
        />
      </div>
    </el-card>
  </div>
</template>

<script>
import { getAdminOrderList, updateOrderStatus } from '../../api/admin'

export default {
  name: 'OrderManage',
  data() {
    return {
      // 所有订单数据
      allOrders: [],
      // 状态筛选
      statusFilter: '',
      // 过滤后的订单
      filteredOrders: [],
      // 分页
      currentPage: 1,
      pageSize: 10,
      // 订单状态文本映射（统一：0待付款/1已付款/2已发货/3已完成/4已取消）
      statusTextMap: {
        0: '待付款',
        1: '已付款',
        2: '已发货',
        3: '已完成',
        4: '已取消'
      },
      // 订单状态标签颜色映射
      statusTypeMap: {
        0: 'warning',
        1: 'primary',
        2: 'danger',
        3: 'success',
        4: 'info'
      }
    }
  },
  computed: {
    // 当前页数据
    pagedOrders() {
      const start = (this.currentPage - 1) * this.pageSize
      return this.filteredOrders.slice(start, start + this.pageSize)
    }
  },
  methods: {
    // 加载订单列表
    async loadOrders() {
      try {
        const result = await getAdminOrderList()
        if (result.code === 200) {
          this.allOrders = result.data || []
          // 按创建时间降序排序
          this.allOrders.sort((a, b) => new Date(b.createTime) - new Date(a.createTime))
          this.filterOrders()
        } else {
          this.$message.error('加载订单列表失败：' + (result.message || ''))
        }
      } catch (error) {
        console.error('加载订单列表失败：', error)
        this.$message.error('网络错误，请稍后重试')
      }
    },

    // 状态筛选
    filterOrders() {
      if (this.statusFilter === '') {
        this.filteredOrders = [...this.allOrders]
      } else {
        const status = Number(this.statusFilter)
        this.filteredOrders = this.allOrders.filter(o => o.status === status)
      }
      this.currentPage = 1
    },

    // 格式化时间
    formatTime(timeStr) {
      if (!timeStr) return '--'
      const d = new Date(timeStr)
      if (isNaN(d.getTime())) return timeStr
      const year = d.getFullYear()
      const month = String(d.getMonth() + 1).padStart(2, '0')
      const day = String(d.getDate()).padStart(2, '0')
      const hour = String(d.getHours()).padStart(2, '0')
      const minute = String(d.getMinutes()).padStart(2, '0')
      return year + '-' + month + '-' + day + ' ' + hour + ':' + minute
    },

    // 更新订单状态（发货/完成）
    async handleUpdateStatus(row, status) {
      const statusText = status === 2 ? '发货' : '完成'
      try {
        await this.$confirm(`确定要${statusText}该订单吗？`, '确认操作', {
          confirmButtonText: '确定',
          cancelButtonText: '取消',
          type: 'warning'
        })

        const result = await updateOrderStatus({ id: row.id, status })
        if (result.code === 200) {
          this.$message.success(`订单${statusText}成功`)
          this.loadOrders()
        } else {
          this.$message.error(result.message || '操作失败')
        }
      } catch {
        // 用户取消
      }
    }
  },
  created() {
    this.loadOrders()
  }
}
</script>
```

## 12.4 AdminStatsController补充

> "图表数据需要后端接口提供。我们在AdminStatsController中新增两个统计接口：近7天订单趋势和分类销售额。注意要使用LocalDate正确比较日期，分类销售用SQL的JOIN+GROUP BY实现。"

### 步骤3：完整的AdminStatsController

创建文件 `backend/src/main/java/com/ecommerce/controller/AdminStatsController.java`：

```java
package com.ecommerce.controller;

import com.ecommerce.mapper.OrderMapper;
import com.ecommerce.mapper.ProductMapper;
import com.ecommerce.mapper.UserMapper;
import com.ecommerce.service.UserService;
import com.ecommerce.util.Result;
import jakarta.servlet.http.HttpServletRequest;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.*;

import java.time.LocalDate;
import java.time.format.DateTimeFormatter;
import java.util.ArrayList;
import java.util.HashMap;
import java.util.List;
import java.util.Map;

@RestController
@RequestMapping("/api/admin/stats")
public class AdminStatsController {

    @Autowired
    private ProductMapper productMapper;

    @Autowired
    private OrderMapper orderMapper;

    @Autowired
    private UserMapper userMapper;

    @Autowired
    private UserService userService;

    private Result checkAdmin(HttpServletRequest request) {
        Integer userId = (Integer) request.getAttribute("userId");
        if (!userService.checkAdmin(userId)) {
            return Result.error(403, "无权限");
        }
        return null;
    }

    /**
     * 数据概览
     * GET /api/admin/stats/overview
     */
    @GetMapping("/overview")
    public Result overview(HttpServletRequest request) {
        Result adminCheck = checkAdmin(request);
        if (adminCheck != null) return adminCheck;

        Map<String, Object> data = new HashMap<>();
        data.put("productCount", productMapper.count());
        data.put("todayOrderCount", orderMapper.countToday());
        data.put("totalSales", orderMapper.sumPaidTotalPrice());
        data.put("userCount", userMapper.count());
        return Result.success(data);
    }

    /**
     * 近7天每日订单数
     * GET /api/admin/stats/orderTrend
     * 返回：[{date: "2025-01-01", count: 5}, ...]
     */
    @GetMapping("/orderTrend")
    public Result orderTrend(HttpServletRequest request) {
        Result adminCheck = checkAdmin(request);
        if (adminCheck != null) return adminCheck;

        LocalDate today = LocalDate.now();
        DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd");
        List<Map<String, Object>> result = new ArrayList<>();

        for (int i = 6; i >= 0; i--) {
            LocalDate date = today.minusDays(i);
            String dateStr = date.format(formatter);
            long count = orderMapper.countByDate(dateStr);
            Map<String, Object> item = new HashMap<>();
            item.put("date", dateStr);
            item.put("count", count);
            result.add(item);
        }
        return Result.success(result);
    }

    /**
     * 分类销售额统计
     * GET /api/admin/stats/categorySales
     * 按product.category分组求和total_price
     * 返回：[{category: "鲜花", sales: 99999.00}, ...]
     */
    @GetMapping("/categorySales")
    public Result categorySales(HttpServletRequest request) {
        Result adminCheck = checkAdmin(request);
        if (adminCheck != null) return adminCheck;

        List<Map<String, Object>> list = orderMapper.selectCategorySales();
        return Result.success(list);
    }
}
```

> "注意几个关键修复点：第一，每个方法都添加了 `checkAdmin()` 权限校验，只有管理员才能访问统计接口；第二，不再使用 JdbcTemplate，所有统计查询都通过 Mapper 方法实现；第三，`categorySales` 的 JOIN+GROUP BY 查询在 OrderMapper.xml 中定义。"

    private ProductMapper productMapper;

1. **Dashboard首页概览**：4个统计卡片（商品总数、今日订单、总销售额、用户总数）+ 快捷操作，从后端接口 `GET /api/admin/stats/overview` 获取数据。统计卡片颜色为花之恋品牌色：#C71526/#E8475F/#FF6B81/#D4A574。

2. **商品管理页面**：用 Element Plus 的 `el-table` 展示商品列表，实现了新增（`el-dialog`+`el-form`）、编辑、删除（`this.$confirm`）、上架/下架切换功能，带搜索和 `el-pagination` 分页。商品分类选项更新为：鲜花/永生花/花束/花篮/绿植/礼品花。

3. **订单管理页面**：用 Element Plus 的 `el-table` 展示订单列表，实现了按状态筛选（`el-select`）、发货（改状态为2）、完成（改状态为3）功能，带分页。

4. **AdminStatsController**：注入ProductMapper+OrderMapper+UserMapper+UserService，每个方法都有checkAdmin()权限校验。categorySales返回的分类销售额按花之恋分类（鲜花/花束/永生花/绿植/花篮/礼品花）统计。

5. **Mapper层**：ProductMapper和OrderMapper方法命名遵循统一规范（find/count/search前缀用于查询，update前缀用于修改），不使用selectAll/updateById/deleteById等通用名。ProductMapper包含15个方法覆盖前台列表、后台管理、搜索、下单等场景。

6. **测试数据**：提供了花之恋鲜花电商相关的测试商品数据（红玫瑰、康乃馨、永生花礼盒、向日葵花束、开业花篮、多肉植物等），确保系统运行效果贴合鲜花电商主题。
